# Frontend (React + Vite)

This is the frontend application for the Razor-Pay demo, built with React and Vite.

**Quick Start**

- **Install dependencies:** Run `npm install` in the `frontend` folder.
- **Run in development:** `npm run dev` — starts Vite with HMR on http://localhost:5173 by default.
- **Build for production:** `npm run build` — outputs static files to `dist`.
- **Preview production build:** `npm run preview` — serves the built files locally.

**Notes**

- This project uses `axios` for API requests. The backend server is located in the `backend` folder of the workspace.
- ESLint is configured; run `npm run lint` to check code style.

If you need help wiring the frontend to the backend (CORS, base URLs, or environment variables), tell me which environment you want and I can add an example `.env` setup.
